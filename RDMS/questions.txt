5)Why RDMS prefered over file based system 
    1️ Reduced Data Redundancy
    File System: Same data is stored in multiple files → duplication.
    RDBMS: Data is stored in tables with relationships → reduces duplication.

    2️ Better Data Consistency
    File System: Updating data in one file may not update other files → inconsistency.
    RDBMS: Changes reflect everywhere due to centralized database.

    3️ Improved Data Security
    File System: Limited security control.
    RDBMS: Provides user authentication, roles, and access control.

    4️ Data Integrity
    File System: No strict rules for maintaining accuracy.
    RDBMS: Uses constraints (Primary Key, Foreign Key) to maintain correct data.

    5️ Easy Data Access
    File System: Complex and time-consuming to retrieve data.
    RDBMS: Uses SQL queries for fast and flexible data retrieval.

    6️ Multi-User Access
    File System: Difficult to handle multiple users simultaneously.
    RDBMS: Supports concurrent access with transaction management.
